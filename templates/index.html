Stremio Catalogs

By Justin

About

Stremio Catalogs is a Stremio addon that adds more catalogs for movies genres on the homescreen. It aims to make the experience more like netflix.

Installation

To install, follow these steps.

  1. Copy this link address. (Make sure it ends in manifest.json)
  2. Open the Stremio app and go to addons.
  3. Click add a new addon.
  4. Paste the previous link into addon URL.
  5. Click add.
